# Theory:
- Pandas is a powerful Python library used for data manipulation and analysis. It provides easy-to-use data structures like Series and DataFrame for handling
  structured data.
- Main data structures in pandas-Pandas mainly provides two data structures:Series-A Series is a one-dimensional labeled array (similar to a column in a table).
- The keyword as pd creates a short alias pd, which is used to access pandas functions easily.Itstores in the series, and pandas automatically assigns index
  values 0, 1, 2, 3.
- Features of Pandas-easy data cleaning, handling missing values, data filtering and sorting, reading files (CSV, Excel, JSON), statistical analysis and fast data    processing.
- A Series is a one-dimensional labeled array capable of holding data such as integers, floats, or strings.
- A DataFrame is a two-dimensional data structure similar to a table with rows and columns. It is commonly used to store and analyze datasets.
- Import pandas as pd → Used to import the Pandas library for data manipulation and analysis.
## Pandas allows users to perform many operations on datasets such as: 
- pd.Series() → Used to create a Series, which stores one-dimensional labeled data.  
- pd.DataFrame() → Used to create a DataFrame, which stores data in tabular (row–column) form.  
- print() → Used to display the Series and DataFrame output.  
- .shape → Used to find the number of rows and columns in the DataFrame.  
- .ndim → Used to find the number of dimensions of the DataFrame.  
- .size → Used to find the total number of elements in the DataFrame.  
- .columns → Used to display the column names of the DataFrame.  
- .dtypes → Used to check the data type of each column.  
- df['column_name'] → Used to access a specific column from the DataFrame.  
- .loc[] → Used to access data using row labels and column names.  
- .iloc[] → Used to access data using row and column positions.  
- df['new_column'] = values → Used to add a new column to the DataFrame.  
- df.at[] / df.loc[] = value → Used to update specific values in the DataFrame.  
- df.drop() → Used to delete unwanted columns from the DataFrame.   
- .mean(), .min(), .max() → Used to perform basic statistical operations on numerical data.  
- df[condition] → Used to filter data based on a given condition.
## Applications of Pandas Library in EDA:
- Data Manipulation and Analysis
Pandas is used to organize, analyze, and summarize data using structures like Series and DataFrame.
- Data Cleaning and Preparation
  Pandas helps in handling missing values, filtering data, and modifying datasets before performing analysis.
- Applications of Pandas-data analysis, machine learning preprocessing, financial data analysis, scientific research and business analytics.
